# Трекер настроения и продуктивности (Telegram бот)

Бот помогает отслеживать ежедневное настроение, часы работы/учебы и сон, а затем анализирует данные и выдаёт персонализированные инсайты.

## Возможности

- Запись данных за день: настроение (1–5), продуктивные часы, сон, комментарий.
- Просмотр статистики за неделю/месяц.
- Генерация графика динамики показателей.
- Анализ связей между сном и продуктивностью, работой и настроением.
- Команды: `/start`, `/add`, `/stats`, `/history`, `/clear`, `/help`.
- Удобные клавиатуры (Reply и Inline).

## Технологии

- Python 3.10+
- pyTelegramBotAPI
- PostgreSQL
- psycopg2
- matplotlib, pandas, numpy
